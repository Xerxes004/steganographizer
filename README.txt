


options:
	Encrypt the given image with a message from the command
	line, or from the optional input file.
	-e <original image> <modified image> [optional input file]

	Decrypt the given image. The message is written to the
	console, or stored in the optional output file.
	-d <modified image> [optional output file]

	Analyze the given bmp file
	-a <image.bmp>

	-h show this help
